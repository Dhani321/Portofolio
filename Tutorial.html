Tutorial Deploy Website ke GitHub

Langkah 1: Siapkan Repository di GitHub

  1. Buka GitHub dan login ke akun Anda.
  2. Klik tombol New untuk membuat repository baru.
  3. Masukkan nama repository (misalnya, my-website).
  4. Pilih Public agar dapat diakses semua orang.
  5. Klik Create repository untuk membuat repository.

Langkah 2: Upload File Website ke Repository

  • Menggunakan Git:

Jika sudah menginstal Git, buka terminal dan jalankan perintah berikut:

                          
          git init
          git add .
          git commit -m "Initial commit"
          git branch -M main
          git remote add origin https://github.com/username/my-website.git
          git push -u origin main

Ganti username dan my-website sesuai dengan nama pengguna dan repository Anda.

Langkah 3: Aktifkan GitHub Pages

  1. Masuk ke halaman repository Anda di GitHub.
  2. Klik tab Settings.
  3. Gulir ke bawah dan pilih Pages.
  4. Pilih main (atau master) sebagai branch sumber GitHub Pages.
  5. Klik Save untuk menyimpan pengaturan.
  6. GitHub akan memberikan URL untuk website Anda, misalnya: https://username.github.io/my-website.

Langkah 4: Cek Status Website

  1. Tunggu beberapa menit hingga GitHub Pages memproses website Anda.
  2. Buka URL yang diberikan oleh GitHub Pages untuk melihat website Anda online.

Langkah 5: Update Website (Opsional)

Jika ingin memperbarui konten website, lakukan perubahan pada file di proyek lokal Anda, lalu jalankan perintah berikut:

                          
          git add .
          git commit -m "Update website content"
          git push origin main

Perubahan akan otomatis diperbarui di GitHub Pages setelah beberapa saat.
