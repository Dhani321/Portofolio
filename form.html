Penjelasan Pembuatan Form Login

Berikut adalah tutorial pembuatan form login menggunakan HTML berdasarkan kode yang Anda berikan:

<div style="padding-top: 90px; display: flex; justify-content: center">
<!-- Container utama untuk form login -->
<div class="login-container">
<h2>Login</h2>
<!-- Form login, method POST digunakan untuk mengirim data secara aman -->
<form action="/login" method="POST">
<!-- Input username -->
<label for="username">Username:</label>
<input type="text" id="username" name="username" required />
<!-- Input password -->
<label for="password">Password:</label>
<input type="password" id="password" name="password" required />
<!-- Tombol submit dan reset -->
<div class="button-group">
<input type="submit" value="Login" />
<input type="reset" value="Reset" />
</div>
</form>
</div>
</div>

Langkah-langkah Pembuatan Form Login

1. Buat Struktur Dasar HTML

Langkah pertama adalah membuat file HTML dengan struktur dasar. Pastikan file ini dimulai dengan deklarasi <!DOCTYPE html> dan tag pembuka serta penutup HTML.

2. Membuat Container Utama

Untuk membuat form login berada di tengah halaman, kita menggunakan elemen <div> dengan properti CSS display: flex; dan justify-content: center;. Ini memposisikan konten form di tengah secara horizontal.

3. Menambahkan Form Login

Form login dibuat menggunakan elemen <form> dengan metode POST. Input untuk username dan password dibuat dengan elemen <input>, serta menggunakan atribut required untuk memastikan kolom tersebut harus diisi.

4. Menambahkan Tombol Submit dan Reset

Dua tombol dibuat menggunakan elemen <input>, satu untuk submit dan satu lagi untuk reset. Tombol-tombol ini membantu pengguna untuk mengirim atau mereset data yang telah dimasukkan ke dalam form.

5. Kesimpulan

Dalam tutorial ini, Anda telah belajar cara membuat form login sederhana dengan HTML dan sedikit styling CSS untuk memperbaiki tata letaknya. Anda bisa menambahkan lebih banyak styling untuk memperindah form ini.

Pada form login ini, elemen HTML <form> digunakan untuk menangkap dan mengirimkan data input dari pengguna. Atribut action="/login" menunjukkan bahwa data yang dimasukkan akan dikirim ke URL "/login" saat pengguna mengirimkan form, sementara atribut method="POST" digunakan untuk mengirimkan data secara aman ke server. Form ini terdiri dari dua input utama, yaitu username dan password. Elemen <label> digunakan untuk memberikan label yang deskriptif kepada setiap input, di mana input username menggunakan elemen <input type="text"> dan input password menggunakan elemen <input type="password">. Keduanya diberi atribut required, yang memastikan bahwa pengguna tidak bisa mengirimkan form tanpa mengisi kedua input tersebut. Di bagian akhir, terdapat dua tombol: tombol "Login" yang dibuat dengan elemen <input type="submit"> untuk mengirimkan form, dan tombol "Reset" dengan elemen <input type="reset"> untuk menghapus semua input yang telah dimasukkan pengguna.
